## 🔧 Troubleshooting

### If you see TypeScript errors:
Run `npm install` first, then restart the dev server.

### If the page doesn't load:
1. Check terminal for errors
2. Make sure port 3000 is not in use
3. Try `npm run dev` again

### If styles don't work:
Make sure Tailwind CSS is properly installed with `npm install`
